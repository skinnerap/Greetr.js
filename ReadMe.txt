Greetr.js

Free Usage Declared for All

Special Note:
Greetr.js comes with jQuery bundled into the minified js file!
This means you instantly get access to all the features of
jQuery with Greetr.js. Greetr.js fully supports jQuery features.

To Use:
1 - Download the source code from the Github Repo
2 - Copy the greetr.min.js file into your project and include it into your script tag first
into your HTML page.
3 - Use Greetr.js easily in your application

Examples of Usage: https://skinnerap.github.io/Greetr.js/
