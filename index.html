Greetr.js

What is Greetr.js?

You can use Greetr.js to easily update any HTML element on the DOM with a variety of greetings to the user. Greetr.js incorporates jQuery functionality, allowing you as a developer to easily create Greetr objects with very little typing, in a familiar format.

Try it Out!

Setup:

const fName = $(Selector1).val();
const lName = $(Selector2).val();
const lang = $(Selector3).val();
const gObj = G$(fName, lName, lang);

Casual Greeting:

gObj.updateHtml(HTML Selector);

Result:

Formal Greeting:

gObj.updateHtml(HTML Selector, true);

Result:

Change Language:

gObj.setLang('es');

Result: Changes the language to Spanish

Current Supported Languages:

  • English: 'en'
  • Spanish: 'es'
  • German: 'de'
  • Korean: 'ko'
  • Italian: 'it'
Change Language and Casual Greet:

gObj.setLang('es').updateHtml(Html Selector);
// Could use the formal version by setting the second parameter
// of the updateHtml function as 'true'

Result:

Change Language and Formal Greet:

gObj.setLang('es').updateHtml(Html Selector, true);
